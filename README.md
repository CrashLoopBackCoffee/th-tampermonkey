# th-tampermonkey
Useful Tampermonkey Scripts

## Scripts

### copy-pr-title
Adds a button to GitHub PR pages to copy the PR title and URL to the clipboard.

**Install:** [copy-pr-title.user.js](copy-pr-title/copy-pr-title.user.js)

## Workflows

- **`/approve`** — comment on any PR to approve and squash-merge it
- **Auto-merge** — new PRs automatically get auto-merge enabled (non-draft only)
